## 🛍️ 서비스 소개

<img width="911" height="512" alt="2" src="https://github.com/user-attachments/assets/0a53d90d-074e-45f9-80c3-6200a2cfe455" />

> **지역 기반 매칭 + 소분 거래 시스템 + AI 추천**을 결합하여  
> 누구나 신뢰할 수 있는 공동 소비 경험을 제안하는 플랫폼입니다.

**PORTIONY**는 대용량 제품을 혼자 구매하기 부담스러운 소비자들이  
필요한 만큼만 나눠 가질 수 있도록 기획된 *소분 기반 공동소비 플랫폼*입니다.

본 프로젝트는 **생성형 AI 기반의 글로벌 서비스 구축**이라는 주제에서 출발하여,  
생성형 AI 기술이 실생활의 소비 문제에 어떻게 적용될 수 있을지를 고민하였습니다.  
그 결과, 대용량 상품을 나눠 쓰는 **소분 거래**와 이를 **지역 커뮤니티**와 연결하는  
**공동 소비 서비스**라는 방향으로 아이디어를 확장하게 되었습니다.

서비스명 **Portiony**는  
portion(소분하다, 나누다)와 community, harmony의 의미를 결합해 탄생한 이름입니다.  
**필요한 만큼 사고, 남는 건 나누는** 소비 문화를 제안하는 것이 목표입니다.

**지역 기반 매칭**, **소분 거래 시스템**, **AI 추천 기술**을 결합하여  
보다 신뢰할 수 있고, 사람 간 연결이 살아있는 소비 경험을 지향합니다.

## 🔧 기술 스택

<img width="911" height="512" alt="8" src="https://github.com/user-attachments/assets/a378a082-0b56-4f33-bacf-8f60b4c3a19e" />

### Frontend  
- React  
- React Router  
- Axios  
- CSS Modules  

### Backend  
- Spring Boot  
- JPA / MySQL  
- JWT 인증 방식  

### Deployment  
- Netlify (프론트엔드 배포)  
- Cloudtype (백엔드 배포)
